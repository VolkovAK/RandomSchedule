# RandomSchedule
Телеграм-бот, генерирующий случайное время прихода на следующий день

Все времена, дедлайны и отложенные сообщения — по часовому поясу **Europe/Moscow**.


### Команды

1) `/time`, `/random` — время на завтра (обычный / туман войны / царь горы)
2) `/solo` — персональный вызов автора команды (без тумана/царя/ставки); вовремя + `/checkin` = страховка
3) `/checkin` — только после `/solo`, только автор соло, до дедлайна
4) `/save` — 50% отменить проигрыш (только владелец страховки, сгорает после использования)
5) `/duel <имя1> <имя2>` — дуэль: кто раньше — победил
6) `/king` — царь горы: после команды — фото с паролем или кружок
7) `/help` — памятка
8) `/get_config` — настройки и владелец страховки
9) `/set_config from 09:00 to 13:00 mean 11:00 sigma 45`
10) `/reset` — сброс расписания (страховка сохраняется)

### Страховка

- В соло-режиме: `/checkin` до дедлайна → страховка переходит к игроку из `/solo`
- Одновременно страховка только у одного человека
- `/save` — 50% шанс отменить проигрыш; после использования страховка сгорает
- Владелец виден в `/get_config`

### Один режим в день

За день — только одно из: `/time`, `/solo`, `/duel`. Повтор той же команды — можно.

### Режимы `/time` и `/solo`

- **Обычный** (50% при дефолте) — точное время
- **Туман войны** (30%) — в чате `~центр ± 2σ`; фактическое время — gauss(центр, σ); Декан сам назовёт точный час
- **Царь горы** (20%) — `/king` + фото/кружок; дедлайн не раньше `from`

### Ставка

При первом `/time` за день: 80% ×1, 15% ×2, 5% ×3. При ×2/×3: `Ставка: xN` в конце. В `/solo` ставки нет.
